## 概要

悪魔の書（抄）は CardWirth シナリオ『悪魔の書』で提供される手動診断機能の再現を目指した Web アプリケーションです。  
Mantine Next.js template を参考に開発されています。

## ローカル環境構築

```sh
# 1. 任意の方法で Node.js をインストール

# 2. 依存関係をインストール
yarn install

# 3. アプリケーションを実行
yarn dev
```
